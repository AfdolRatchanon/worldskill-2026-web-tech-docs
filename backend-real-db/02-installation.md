# บทที่ 2 — ติดตั้งเครื่องมือ

> **บทนี้เตรียมอะไร:** ติดตั้ง 3 อย่างที่ต้องมีก่อนเริ่ม — Node.js (รันโค้ด), MariaDB (ฐานข้อมูล), Postman (ยิงทดสอบ API) — และเช็กว่าใช้งานได้จริง

## 1. Node.js

ดาวน์โหลดเวอร์ชัน LTS จาก [nodejs.org](https://nodejs.org) แล้วเช็ก:

```bash
node -v
npm -v
```

เห็นเลขเวอร์ชันทั้งคู่ = พร้อม (ต้องการ Node 18 ขึ้นไป)

## 2. MariaDB

ติดตั้ง MariaDB (หรือ XAMPP ที่มี MariaDB มาให้) แล้วเช็กว่ารันอยู่ที่ port **3306**

```bash
mysql -u root -p
```

ถ้าเข้า prompt `MariaDB [(none)]>` ได้ = เซิร์ฟเวอร์ทำงาน

::: tip รหัส root ว่าง
ในเครื่องฝึกซ้อมส่วนใหญ่ root ไม่มีรหัส (เคาะ Enter ผ่าน) — ค่านี้จะไปตั้งใน `.env` บทถัดไป (`DB_PASSWORD=`)
:::

## 3. Postman

ดาวน์โหลด [Postman](https://www.postman.com/downloads/) สำหรับยิงทดสอบ API — เราจะใช้ตั้งแต่บทที่ 4 เป็นต้นไป (login, GET/POST endpoint ต่างๆ)

> ในห้องแข่งจริงไม่มีเน็ต — Postman เวอร์ชัน desktop ใช้ออฟไลน์ได้ ติดตั้งไว้ล่วงหน้า

## เช็กความพร้อม

| เครื่องมือ | คำสั่งเช็ก | ต้องเห็น |
|-----------|-----------|---------|
| Node.js | `node -v` | v18+ |
| npm | `npm -v` | เลขเวอร์ชัน |
| MariaDB | `mysql -u root -p` | เข้า prompt ได้ |
| Postman | เปิดโปรแกรม | หน้าต่างพร้อมยิง request |
