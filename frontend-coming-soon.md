# Frontend — กำลังพัฒนา

::: warning กำลังพัฒนา
เนื้อหาส่วน Frontend กำลังอยู่ในขั้นตอนการเขียน จะพร้อมให้ใช้งานเร็วๆ นี้
:::

## ภาพรวม Frontend ที่จะครอบคลุม

| บท | หัวข้อ |
|----|--------|
| F1 | เตรียม Frontend Project (React + Vite + Tailwind CSS) |
| F2 | Authentication และ Context API |
| F3 | Protected Routes และ Role-based Navigation |
| F4 | หน้า Candidate (Submit, My Result) |
| F5 | หน้า Judge (Session Control, Candidates, Recheck, Confirm) |
| F6 | หน้า Manager (Statistics, Export Report) |

## Tech Stack

| Technology | Version | หน้าที่ |
|-----------|---------|--------|
| React | 18+ | UI framework |
| Vite | 5+ | build tool และ dev server |
| Tailwind CSS | 4+ | utility-first CSS |
| Axios | 1+ | HTTP client เรียก API |
| React Router | 6+ | client-side routing |

## ระหว่างรอ Frontend

Backend ที่เรียนมาตั้งแต่บทที่ 1–11 ทำงานได้ครบสมบูรณ์แล้ว ทดสอบได้ผ่าน Postman ทุก endpoint

สามารถ clone frontend จาก repository ของโปรเจ็คและรันควบคู่กับ backend ได้ทันที
